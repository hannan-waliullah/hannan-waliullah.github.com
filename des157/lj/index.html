Response 1: Form Design

21 Jan 2021

When designing a form, designers should strive to make it easy and approachable for the user. In the article “Best Practices for Form Design”, author Salim Ansari lists 20 different considerations when creating an interactive form. It’s important to keep in mind all of the different aspects of the form, from the writing of the input fields to the visual hierarchy of the form. When writing a form, we should eliminate unnecessary questions and gradually increase the complexity of questions. We should avoid using negative language and blaming the user.

Additionally, we have to consider the user interface of the form. This includes placement of the text, alignment, size, color, etc. The author points out smaller things we can design for in order to enhance the user experience. For example, highlighting the first input box can make it easier for the user to locate it. It’s important for the form to be accessible to all users. Accessibility of the form includes proper contrast, text size, and other considerations.
